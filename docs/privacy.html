Privacy Policy

\

Last updated: April 28, 2026

\ \

Ticket Drop Timer: Race Alerts ("we", "us", "our") is an unofficial event tracker. This Privacy Policy explains what data we collect and how we use it.

\ \

1. Information We Collect

\
\

We may collect the following:

\
    \
  • Watchlist and alert preferences you set in the app.
    • \
  • Push notification device token, only if you allow notifications.
    • \
  • Public event/ticket page metadata used to display event status.
    • \
  • Basic technical logs needed to keep the service reliable.
    • \
\

We do not sell tickets, process payments, or collect payment card data.

\
\ \

2. How We Use Information

\
    \
  • Show event tracking data and countdown/release states.
    • \
  • Provide watchlist features and alert triggers.
    • \
  • Send notifications you have enabled.
    • \
  • Maintain, secure, and improve app reliability.
    • \
\ \

3. Data Sharing

\

We do not sell personal data. We may use service providers for hosting and delivery infrastructure required to run the app.

\ \

4. Data Retention

\

We keep data only as long as needed to operate the app and provide requested features, unless a longer period is required by law.

\ \

5. Your Choices

\
    \
  • You can disable notifications at any time in iOS Settings.
    • \
  • You can remove watchlist items from within the app.
    • \
  • You can request support or deletion inquiries by email.
    • \
\ \

6. Third-Party Links

\

The app may open external event or ticket pages. We are not responsible for third-party privacy practices.

8. Unofficial Status
